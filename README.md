# Exploring-and-Visualizing-a-Simple-Dataset 


A simple analysis of the Iris dataset focusing on structure understanding and visual exploration.

Main Steps

Loaded dataset with seaborn.

Inspected basic structure (shape, columns, info, describe).

Created scatterplot, pairplot, histograms, and boxplots.

Visualized feature relationships and species separation.

Tools

Python, pandas, seaborn, matplotlib
